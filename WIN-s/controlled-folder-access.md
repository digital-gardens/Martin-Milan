# Controlled folder access
Controlled folder access is a security feature in Windows that helps protect your files and folders from unauthorized changes by malicious software. When enabled, it prevents untrusted applications from making changes to protected folders, which can help safeguard your important data.

To enable controlled folder access, follow these steps:

win + q -> controlled folder access -> Manage controlled folder access -> Turn on controlled folder access

win + s protected folders -> Manage protected folders -> Add a protected folder
