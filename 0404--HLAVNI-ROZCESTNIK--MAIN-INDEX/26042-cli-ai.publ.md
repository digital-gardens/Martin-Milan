### GitHub Copilot in VSCode

You can use GitHub Copilot to generate CLI commands directly in your code editor (VSCode). Simply start typing a command, and Copilot will suggest completions based on the context. You can then review and approve the suggested commands before executing them in the terminal.

### Polyglot Notebooks in VSCode
You can use Polyglot Notebooks in VSCode to write and execute PowerShell commands. You can ask AI to generate commands in the notebook, and then execute them directly within the notebook environment. This way, you can see the output of the commands and ask AI to analyze the results without needing to copy and paste anything.
